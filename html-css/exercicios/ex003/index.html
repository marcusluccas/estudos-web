Usando umas Imagens

A Html para imagens

O comando para usar imagens em html é <img scr="Arquivo.png ou a url de uma imagem" alt"Texto alternativo">.

As Css para imagens

As Css são usadas para alterar as posições das imagens e tamanho.

O Javascripts para imagens

O Javascript são usando para fazer menus com imagens.
